release notes rapiddweller-benerator-ce 1.2.0
==============================================

summary
-------
Benerator CE 1.2.0 enhances performance and documentation and fixes some bugs

enhancements
------------
- Improved Benerator engine performance by 20%
- Improved and extended Benerator Manual
- Added address domain package for United Kingdom (GB)
- Supporting custom random implementations

changes
-------
- Redesigned the Distribution class hierarchy, introduced the AbstractDistribution
  and DetachedSequence classes to compensate this for implementors. Se the manual's
  migration guide what to do if you programmed custom Distributions.
